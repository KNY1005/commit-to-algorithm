[자체문제]

벨만-포드 알고리즘을 구현한 solution()함수를 구현하세요. gragh의 각 데이터는 배열입니다. 첫 번째 데이터는 0번 노드를 기준으로 연결된 노드 정보. 두번째 데이터는 1번 노드를 기준으로 연결된 노드 정보 입니다. 노드 정보의 구성은 (노드, 가중치)와 같습니다. source는 시작노드 입니다. 반환값은 최단 거리를 담은 distance 배열과 최단 거리와 함께 관리할 직전 노드 presecessor를 배열에 차례대로 담아서 [distance, presecessor] 형식으로 반환하면 됩니다. presecessor에서 시작 노드는 null로 표시합니다. 만약 음의 가중치 순회가 있다면 [-1]을 반환하세요. 음의 가중치 순화랑 순환하면 할수록 가중치의 합이 적어지는 순회를 말합니다.

## 제약 조건
- 노드의 최대 개수는 100개 입니다.
- 각 간선의 가중치는 -100 이상 100 이하 입니다.

## 입출력의 예
graph
[[(1, 4). (2, 3), (4, -6)], [(3, 5)], [(1, 2)], [(0, 7), (2, 4)], [(2, 2)]]

source
0

return
[[0, -2, -4, 3, -6], [null, 2, 4, 1, 0]]
//-------------------------------

graph
[[(1, 5), (2, -1)], [(2, 2)], [(3, -2)], [(1, 6)]]

source
0

return
[-1]
